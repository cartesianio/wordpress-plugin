# Cartesian Agent WordPress Plugin

Integrates [Cartesian's](https://cartesian.io) AI-powered agent into WordPress admin pages for enhanced user experience and personalization.

## Getting Started

1. Go to [home.cartesian.io](https://home.cartesian.io/), sign in with your email, and generate an **API Key**
2. Download the latest `cartesian-agent.zip` from the [Releases](https://github.com/cartesianio/wordpress-plugin/releases) page
3. In your WordPress admin, go to **Plugins > Add New > Upload Plugin**
4. Upload the downloaded ZIP file and click **Install Now**
5. Activate the plugin
6. Navigate to **Settings > Cartesian Agent**
7. Enter the API key under the **Agent Key** field
8. Toggle **Enable agent tracking**
9. Click **Save**

## Requirements

- WordPress 6.0 or later
- PHP 8.0 or later
- A [Cartesian](https://cartesian.io) account
